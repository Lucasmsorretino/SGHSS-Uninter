# SGHSS - Sistema de Gestão Hospitalar (VidaPlus)

## Stack Tecnológica
* **Linguagem:** Python 3.10+
* **Framework Backend:** Flask (sugerido pela leveza) ou Django.
* **Banco de Dados:** SQLite (dev) / PostgreSQL (prod).
* **Testes:** Pytest / Unittest.

## Estrutura do Projeto
* `/src`: Código fonte da aplicação (Controllers, Models).
* `/docs`: Diagramas (DER, UML) e Documentação PDF.
* `/tests`: Testes automatizados (Unitários e Integração).
* `requirements.txt`: Dependências do Python.
